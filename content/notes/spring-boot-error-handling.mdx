## 一个简单可用的策略

对于一个早期项目来说，通常至少要做到下面三点：

1. validate request input at the edge
2. throw domain-oriented exceptions in service code
3. map them to stable response shapes in one place

```java
@RestControllerAdvice
public class GlobalExceptionHandler {
  @ExceptionHandler(IllegalArgumentException.class)
  public ResponseEntity<ApiError> handleBadRequest(IllegalArgumentException exception) {
    return ResponseEntity.badRequest().body(new ApiError("BAD_REQUEST", exception.getMessage()));
  }
}
```
